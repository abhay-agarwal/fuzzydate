A flexible date parser library for Rust.

## Usage

Put this in your `Cargo.toml`:

```toml
[dependencies]
fuzzydate = "0.2"
```

## Example

```rust
use fuzzydate::parse;

fn main() {
    let input = "five days after this friday";
    let date = parse(input).unwrap();
    println!("{:?}", date);
}
```
